LESSON:    Retrieving Some Data from a Table 

You can restrict the number of columns shown
in a Result Set by specifying the column names
you wish to see.

Example...

  SELECT title, price
  FROM book;


This shows ALL the rows stored in the book table,
but only the title and price information.








Let's look at another example...

  SELECT name, city, state
  FROM publisher;


Here you see all the rows from the publisher table,
but only the name, city, and state information.
