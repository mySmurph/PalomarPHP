LESSON:    The MIN and MAX Functions 

There are two functions that find the least or greatest...

  MIN(column_name)		- Returns the minimum value found in the
                          			  column for all rows in the Result Set

  MAX(column_name)		- Returns the maximum value found in the
                          			  column for all rows in the Result Set

The MIN and MAX functions work with all datatypes, such
as numbers, text, and dates.




For a number, the MIN returns the lowest numerical value found
in the specified column for set of rows returned by the SELECT
statement.   The MAX returns the highest numerical value.

For text information, the MIN returns the lowest ASCII order value
found in the specified column for set of rows returned by the SELECT
statement.   The MAX returns the highest ASCII order value.  ASCII
order values imply alphabetical order for regular text.

For a date, the MIN returns the earliest calendar value found in the
specified column for set of rows returned by the SELECT statement.
The MAX returns the latest calendar value.




Some Examples...

  SELECT MIN(advance), MAX(advance)
  FROM book;

  SELECT MIN(lastname), MAX(lastname)
  FROM author;

  SELECT MIN(pubdate), MAX(pubdate)
  FROM book;
