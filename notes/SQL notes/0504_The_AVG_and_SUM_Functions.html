LESSON:    The AVG and SUM Functions 

There are two functions that deal with total or average column values...

  SUM(column_name)		- Totals the values in specified column
  						  for all rows in the Result Set

  AVG(column_name)		- Averages the values in specified column
  						  for all rows in the Result Set




The SUM function considers all non-NULL values, for the column
specified, and adds them up.

This function only works with numbers!

It only adds the column values for the rows that pass any
criteria specified in the WHERE clause, etc.

Example...

  SELECT SUM(advance)
  FROM book;


  




The AVG function considers all non-NULL values, for the column
specified, adds them up... and then, divides that total by
the number of rows that were added up.

This function only works with numbers!

It only adds the column values for the rows that pass any
criteria specified in the WHERE clause, etc.

Example...

  SELECT AVG(advance)
  FROM book;
